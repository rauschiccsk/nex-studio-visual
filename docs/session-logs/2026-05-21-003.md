## Workflow disciplína

**Hybrid pracovný postup:**
- Sub-round 1A-1C + 2 + 3 (F-001..F-004): krok-za-krokom outline → write → schválenie → commit
- Sub-round 3 (F-005, F-006) + Sub-round 4: batch write po Director's explicit approval

**Dôvera-based escalation:** Director zaviedol batch approval po pozorovaní konzistentnej kvality cez 9 dokumentov (customer-requirements, customer-dialogue, summary, development-spec, F-001..F-004). Tento pattern ukazuje že Designer output v stabilnom kontexte (jasné Direktorove rozhodnutia v session 2026-05-21-002) môže ísť rýchlejšie.

**Quality preserved:**
- Pre-flight check pre menu of variants (eliminate obviously-bad pred ponukou)
- Quality-first principle (jedno najlepšie riešenie + rationale per resolution)
- Slovenský jazyk s minimum anglicizmov
- Štruktúrované sekcie podľa schváleného outline-u
