## Key decisions

- **Pragmatic CC change pre NEX Studio sebadevelopment** — drobné UI refactor nejde cez Designer/Implementer/Auditor pipeline (overkill). Quality-first kompenzovaný dôraznou špec pred kódom (12 AC, TDD workflow).
- **TDD pre čisto pure funkciu** — `buildTree` testovaný komplet bez UI (11 tests). Component testy s RTL pre KbTree (9 tests). Lacné, vysoký coverage.
- **Unicode emoji namiesto Lucide SVG ikony** — pre folder/file rozlíšenie. Native OS-rendered, farebné, žiadne extra CSS. Lucide chevrony ostávajú (UI indikátor toggle stavu).
- **`selectedCategory` state odstránený úplne** — namiesto `?` fallback alebo session migration. Quality-first: ak setter nie je voláný a categories sidebar zmizol, state je dead code. Čistá drop.
- **2-stĺpcový layout** — tree column `w-80` (20rem) + viewer `flex-1`. Predtým 3 stĺpce (sidebar + categories + documents = 28rem). Viewer získal ~15% plochy.
