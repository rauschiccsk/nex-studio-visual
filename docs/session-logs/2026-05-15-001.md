## Key decisions

- **Deep audit cez paralelný sub-agent dispatch** — 9 sekcií content auditu cez 17 000 r dokumentov bolo overkill pre single-context execution. 4 paralelné sub-agenti (A, B, C, D — najťažšie scans) + ja sekvenčne (E-I — kratšie checks) škrtli wall-clock 2-3 h → 30-40 min. Quality nestratený — každý sub-agent dostal focused scope s konkrétnymi findings buckets (✅/⚠/❌, severity calibration). Worth memory ako pattern.

- **Audit verdict scoring** — Implementer-readiness 8.5/10 founded na: žiadny blocker + 4 infra fixes (~15 min) + 4 spec drobnosti ktoré Designer fix concurrent. Director's "chcem byť presvedčený" satisfied cez triangulated findings (4 sub-agents + 5 sekcií jamou).

- **L-016 lesson aplikované aj na Implementer settings.json** — pôvodne fix targeted len Designer + Auditor (mali deny + bug). Implementer bol "no functional bug" (allow + harmless). Audit F.1 prinútilo dotiahnutie principal "no relative globs anywhere" — defensive precision. Plus belt-and-suspenders explicit deny pre customer-requirements (charter rule bola len Markdown narrative).

- **Sidebar kozmetické iterácie odhalili Director UX preferencie**:
  - **Selected project indicator pri Projects, nie medzi pipeline a agentmi** — locality matters (indicator pri source of action).
  - **Pipeline chronology v sidebar order** — Workflow → Spec → Solution → … bez "vsadenia" non-pipeline entries medzi.
  - **Emoji > line-art glyphs** — "len kontúry, symbolické" je negative; full color glyphs vyhrávajú.

- **Recommendations vs corrections** — pri 3. sidebar iter (reorder) som dal Director "Note pre teba" navrhujúce ďalšie presúvanie Designer/Implementer/Auditor. Director odmietol s vlastným reasoning (logical flow). Quality-first: keď Director explicitly potvrdí stav, žiadne ďalšie "ale možno aj X" suggestions — patrí to do TODO len ak Director sám sa zaujíma. Pre tento prípad: žiadna ďalšia akcia, len acknowledgment.
