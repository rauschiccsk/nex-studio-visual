## Discovery + lessons

1. **F-003 happy-path skeleton anti-pattern** — Generic UAT template predpokladal one shape; reálne target projekty (nex-studio vs nex-inbox) majú heterogeneous per-project variability (DB creds, env passthrough, frontend ports, alembic strategy, crypto formats). CR-022 spec already noted this; CR-023..CR-028 batch potvrdil že dimensional audit pred commit by zachytil >5 z 8 bugov v Designer round.

2. **§10.1 watch-CI-pred-next-commit conflict s rate-limit pri sekvenčných commits** — Implementer pushol 5 commits postupne, sériové `gh run watch` polling exhausted GitHub API rate limit. Lessons learned uložené ako CR-030 Návrh #3 (workflow_dispatch trigger pre recovery) + #4 (stuck-run alert) + relaxed §10.1 interpretation TBD.

3. **§9 self-verification ambiguity** — "lokálne PASS" interpretovateľné ako feature scope (24/24 F-004 tests) ALEBO full suite (2641 tests). CR-030 §9 (e) explicit-eliminuje cez "FULL pytest, NIE selective subset".

4. **nex-inbox repo hygiene gaps** — `.dockerignore` missing pri repo root (context: .) + `psycopg2-binary` mis-classified (alembic je runtime). UAT cross-project smoke vyniesol obidva. Lesson: cross-project verification je legitimate value-add NIE iba tax-on-spec-writing.

5. **Stuck CI run incident** — Single self-hosted runner + GitHub Actions API state inconsistency = recovery limited bez workflow_dispatch escape hatch. CR-030 amendment urobil environment future-resilient.

6. **Director session continuity** — Zoltán's mandate "dnes všetko dokončiť" namiesto session-end mid-day umožnil completion celého v0.2.0 spec balíka v jednej extensive session (8h+ práce, 32 commits naprieč 3 repos).
