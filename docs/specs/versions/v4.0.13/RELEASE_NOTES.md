# v4.0.13 — NEX Studio Visual

## „Nechaj to opraviť" spustí opravu hneď — na jeden klik

Doteraz po kliknutí na „Nechaj to opraviť" build len pripravil opravu a **čakal na druhý klik** („Pokračovať"), aby sa naozaj rozbehol. Nebolo jasné, že treba kliknúť ešte raz — a popri tom svietila mätúca lišta „Znova over bez opravy", hoci si už zvolil opravu.

Od tejto verzie:

- **Jeden klik = oprava beží.** „Nechaj to opraviť" rovno spustí opravu, žiadny medzikrok „Pokračovať".
- Keďže oprava hneď beží, **zmizne aj lišta „Znova over bez opravy"** — už nekonkuruje o pozornosť.

Neexpert tak jedným klikom skutočne opraví, presne ako to čaká.
