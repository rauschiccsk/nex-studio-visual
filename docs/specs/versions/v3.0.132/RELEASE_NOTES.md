## v3.0.132 — Nastavenia po slovensky

### Zrozumiteľné názvy namiesto technických kľúčov
Každé nastavenie v Systéme má teraz ľudský slovenský názov (napríklad „Časový limit generovania návrhovej dokumentácie") namiesto technického kľúča. Pôvodný kľúč ostáva zobrazený malým sivým písmom ako doplnková informácia.

### Popisy po slovensky, ľudskou rečou
Popis každého nastavenia je po slovensky a zrozumiteľne — vieš, čo daná hodnota robí, bez hádania z anglického technického textu.

### Merná jednotka pri každom poli
Za každým editorom je uvedená merná jednotka (sekúnd, minút, správ, € / hod, $ / mil. tokenov, …), takže vždy vieš, v čom hodnotu zadávaš.
