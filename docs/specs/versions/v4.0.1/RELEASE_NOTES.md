# v4.0.1 — NEX Studio Visual

## Dokumenty ukážu aj kontrakty a odovzdávku

Záložka **Dokumenty** doteraz zobrazovala len dokumenty špecifikácie danej verzie. Ostatné dokumenty projektu — **kontrakty** (`docs/contracts/`) a **odovzdávky** pre externých vývojárov (`docs/handoff/`) — v kokpite nebolo vidno; dali sa prečítať len cez Git.

Odteraz sú aj tie priamo v **Dokumentoch**, prehľadne rozdelené do skupín:

- **Špecifikácia** — dokumenty aktuálnej verzie (ako doteraz),
- **Kontrakty** — API kontrakty pre napojené systémy,
- **Odovzdávka** — sprievodné pokyny pre externých vývojárov.

Celý balík, ktorý AI pripraví na odovzdanie, si tak prečítaš priamo v kokpite — bez Gitu.
