## v2.0.0 — Jednoduchší a jednotný základ

### Jeden schopný AI agent namiesto tímu
Systém sme prestavali na štíhlejší základ: prácu teraz vedie jeden schopný AI agent namiesto tímu viacerých rolí. Komunikácia je priamejšia, systém je jednoduchší na údržbu a menej krehký.

### Zjednotené spoločné časti aplikácií
Časti, ktoré majú všetky aplikácie spoločné, sme zjednotili do jedného základu. Vďaka tomu je všetko konzistentnejšie a nové projekty stoja na overenom, jednotnom podloží.
