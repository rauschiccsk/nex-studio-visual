# v4.0.4 — NEX Studio Visual

## Prehľadnejšia a rozumnejšia nezávislá previerka

Dolaďujeme nezávislú previerku návrhu, aby sa lepšie čítala a neopakovala už vyriešené výhrady:

- **Nálezy previerky sa zobrazia ako prehľadný zoznam** — každý bod na vlastnom riadku, nie jeden dlhý blok textu.
- **Previerka neopakuje už vyriešené výhrady.** Keď sa AI a previerka nezhodnú, previerka si pri ďalšom kole svoje predošlé výhrady **znova overí proti aktuálnym dokumentom** a tie vyriešené zahodí — namiesto donekonečna rovnakého zoznamu.
- **Uznáva rozumné riešenia.** Vec vyriešená cez nastavenie s bezpečným predvoleným správaním, vedome odložená s odôvodnením, alebo označená „doladiť pri stavbe" sa už neoznačuje ako „chýba".
