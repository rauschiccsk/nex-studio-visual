# v4.0.26 — NEX Studio Visual

## AI Agent sa pýta zrozumiteľnejšie — vernejšie tvojmu zadaniu

Keď sa ťa AI Agent počas prípravy niečo pýta, robí to teraz tak, aby si vedel rozhodnúť **sám, bez experta**:

- **Drží sa tvojho zadania** — nepýta sa znova na to, čo si už v zadaní rozhodol.
- **Jedno jasné odporúčanie** — namiesto vymyslených ponúk „úzky / stredný / plný" ti dá jednu odporúčanú cestu.
- **Po slovensky, vo výsledkoch** — bez technického žargónu (názvy komponentov, tried, knižníc); popíše, čo to znamená pre aplikáciu a používateľa, nie techniku.

Cieľom je, aby si otázke rozumel a rozhodol o nej priamo v cockpite — bez toho, aby ti ju musel niekto prekladať.
