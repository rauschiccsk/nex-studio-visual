# v4.0.12 — NEX Studio Visual

## Chybu z overenia vyriešiš jedným klikom — bez písania

Keď záverečné overenie nájde chybu, doteraz ti karta niekedy ponúkla len „Usmerniť opravu" — teda **napísať vlastný technický pokyn**. Nešpecialista (junior operátor) to nevie napísať, tak ostal zaseknutý.

Od tejto verzie má karta **vždy jednoklikovú cestu vpred**:

- **„Nechaj to opraviť"** — jedným klikom pošle opravu podľa nálezov overovateľa (Auditora), ten ju spraví a overenie sa zopakuje. **Nemusíš nič písať.**
- „Usmerniť opravu (napíš vlastný pokyn)" ostáva ako **nepovinná** možnosť, keď chceš opravu nasmerovať sám.

Takže build vieš doviesť do konca aj bez experta.
