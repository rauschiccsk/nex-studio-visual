# v4.0.11 — NEX Studio Visual

## Nálezy z overenia sú po slovensky a čitateľné — technický žargón je zbalený bokom

Keď záverečné overenie (Auditor) niečo nájde, doteraz ti napísal hustý technický blok — cesty k súborom, chybové kódy, verzie knižníc, čísla riadkov. Nešpecialista z toho ťažko prečítal podstatu. Doteraz sme Auditorovi len *hovorili*, nech píše ľudsky; nestačilo to.

Od tejto verzie je to riešené **štruktúrou**, nie prosbou:

- **Manažérsky text je krátky a ľudský** — jedna-dve vety: čo nefunguje z pohľadu používateľa a čo treba rozhodnúť. Bez ciest, kódov a žargónu.
- **Všetok technický rozbor je zbalený za „Technický detail"** — jedným klikom si ho rozbalíš, ak ho potrebuješ. Nič sa nestratí, ale bežne ti neprekáža.

Platí pre nález aj pre rozhodovaciu kartu.
