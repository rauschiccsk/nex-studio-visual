## v3.0.175 — Jednotné označenie verzie v nasadeniach

V prehľade nasadení sa verzie zobrazovali nejednotne — raz `v1.0.0` (graduovaná produkčná), raz `1.1.0`. Odteraz sa UAT aj PROD verzie zobrazujú **v rovnakom tvare** (bez „v", napr. `1.0.0` a `1.1.0`). Ide o zjednotenie zobrazenia; samotná uložená hodnota (identifikátor verzie) sa nemení.
