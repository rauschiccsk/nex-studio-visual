## v1.0.0 — Prvá plná verzia workbenchu

### Kompletný vývojový workbench s tímom AI agentov
NEX Studio dozrelo do prvej plnej verzie. Projekt vedie tím špecializovaných AI agentov — jeden navrhuje, jeden programuje, jeden kontroluje — presne ako v skutočnom vývojovom tíme. Celý postup od zadania po hotovú aplikáciu sa premyslí vopred a prebieha pod tvojím dohľadom, takže výsledok je premyslený, nie skladaný za pochodu.
