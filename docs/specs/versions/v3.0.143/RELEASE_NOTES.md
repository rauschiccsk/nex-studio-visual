## v3.0.143 — Jasné tlačidlo „Schváliť na Hotovo"

Keď overenie prešlo a verzia čakala na posledný súhlas manažéra, spodné tlačidlo stále hlásilo „Schváliť plán" a sľubovalo posun do stavby — hoci klik verziu správne uzavrel ako **Hotovo**. Popis teraz zodpovedá skutočnosti: pri overenej verzii sa tlačidlo volá **„Schváliť na Hotovo"** a jasne hovorí, že verziu označí ako hotovú a uzavrie.
