## v3.0.178 — Nasadené appky ukazujú svoju skutočnú verziu

Aplikácie vyvíjané cez NEX Studio zobrazovali v bočnom paneli technickú „build" verziu odvodenú z počtu commitov (`0.1.<číslo>`), ktorá skákala a nesúvisela so skutočnou verziou appky. Odteraz appka ukazuje **svoju skutočnú verziu** (napr. `1.1.0`) — rovnakú ako vo svojich Aktualizáciách. Každá zmena appky = jej vlastná verzia. (NEX Studio samotné, ktoré priebežne opravujeme, si ponecháva plynulé počítadlo.) Prejaví sa pri najbližšom nasadení danej appky.
